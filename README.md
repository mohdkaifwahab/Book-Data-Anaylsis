# 📚 Book Sales Analytics Dashboard

An interactive **Streamlit dashboard** for analyzing book sales data, publishing trends, genres, and author performance.

---

## 🌐 Live Dashboard

You can access the deployed dashboard here:

🔗 **Streamlit App:**
https://book-data-analysis.streamlit.app/

---

## 🚀 Features

* 📊 Key Performance Indicators (KPIs)
* 📈 Multiple Data Visualizations

  * Publishing Year Distribution
  * Books per Genre
  * Sale Price vs Units Sold
  * Units Sold Trend
  * Language Distribution
  * Rating Distribution by Genre
  * Top Authors by Sales
  * Correlation Heatmap
* 📋 Dataset Statistics
* ☑️ Option to view the raw dataset
* ⚡ Optimized data loading using `st.cache_data`

---

## 🛠️ Technologies Used

* Python
* Streamlit
* Pandas
* Matplotlib
* Seaborn

---

## ▶️ Run Locally

Clone the repository:

```bash
git clone https://github.com/kaifwahab001/your-repo-name.git
cd your-repo-name
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the app:

```bash
streamlit run app.py
```
